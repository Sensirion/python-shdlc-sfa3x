Sensirion SFA3x Formaldehyde Sensor Modules Python Driver
=========================================================

This package contains the SHDLC driver for the SFA3x sensor modules which
have an UART interface using the SHDLC protocol.

The driver provides all SHDLC commands as Python methods and handles all the
low-level things like type conversions, checksum calculation, error handling
etc.
